# 什么是多校划片 #

当下绝大部分区域的公办学校，执行的分配规则是单校划片，即一个房子对口一所小学，或一所小学对口一所初中。

而多校划片就是一对多，或多对多。

从概念上不难理解，多校划片改变了资源的分配逻辑，通过不确定性让资源分配看上去更加均衡，以此来削弱由学区房引起的日益高涨的社会矛盾。

我们来看看试行的区域情况如何？

# 多校划片的政策背景 #

国家教育部办公厅每年初会针对全国义务教育，发布当年的招生入学工作通知。

通知中往往会涉及到当年一些重要教育政策，包括民办摇号、集团办学、中考改革、特长生、自主招生等，而“多校划片”就是其中一项频繁被提到的政策。

# 上海多校划片之路——京沪差异 #

北京和上海，一个行政中心，一个经济中心，很多时候都有必要放在一块对比、借鉴。

但要对比，就一定要了解一些背景。

在基础教育领域，北京公办强于民办是众所周知的，我们常常听到几十万一平的天价学区房新闻，也大概率是从北京曝出来的。

相对而言，上海这边要缓和一些，15万/平已经算非常高了，很大原因是上海民办教育普遍强于公办，尤其初中阶段。

![图片](https://picx.zhimg.com/50/v2-f833bc6daa352605a72ca332e32c1353_720w.jpg?source=2c26e567)![图片](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='2999' height='1687'></svg>)造成这种公办民办失衡的主要原因有两点：

1、录取时间先后顺序

2、录取方式的差异

早前，上海民办招生时间是在公办之前，且民办录取方式为“面谈”，而公办是“免试就近入学”。这种政策上的差异简而言之就是，民办通过筛选生源先把好苗子挑走了，这会极大的挫伤整体公办教育的积极性。

2020年的新政改革，改的就是这两点。

录取顺序改成“公民同招”，录取方式改成“民办摇号”。

![图片](https://picx.zhimg.com/50/v2-073b67bf803a9ea7292afe359c61919f_720w.jpg?source=2c26e567)![图片](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='2999' height='1687'></svg>)通过这两点来缓解和改善公办民办失衡的现状，这是上海当下面临的主要矛盾。

那上海多校划片会不会马上大范围推广呢？

# 上海多校划片之路——核心原则 #

我们可以尝试把一些影响较大的政策串联起来看，思路会更清晰。

与其说多校划片，不如说教育均衡，“均衡”大概是诸多政策亘古不变的原则。

1、公办&民办

过去民办强势，现通过公民同招、民办摇号，促进均衡，让“选好学生”变成“教好学生”。

2、择校&不择校

过去小升初择校热，优质生源聚集效应，现通过摇号，降低择校比例，更多人升入到对口初中。

3、鸡头&凤尾

过去中考是优质初中（尤其民办初中）的独角戏，现中考改革名额到校，让普通初中努力的学生也能有更多机会进入重点高中。无法成为凤凰，那就凭实力成为鸡头，照样有机会傲视群雄。

4、稀缺性&家门口

过去优质教育资源极度稀缺，现大力推进学区化、集团化，通过增加供给，让优质教育资源服务更多学生，办成“家门口的好学校”，从根源上缓解供需矛盾。

[上海市推进学区化集团化办学地图](https://link.zhihu.com/?target=http%3A//sd.shlll.net/)![图片](https://picx.zhimg.com/50/v2-ad1991e4cb13e9534080726bb585cfec_720w.jpg?source=2c26e567)![图片](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='2356' height='1482'></svg>)上海学区化集团化办学地图不难看出，在教育均衡化的道路上，政府做了诸多长远考量。

同时，我们也看到上海整个城市规划也是相对均衡的。

我早前统计过一次上海优质小学环线分布情况，如下图：

![图片](https://pic1.zhimg.com/50/v2-bc2641486c7600506d9b5013aef843c0_720w.jpg?source=2c26e567)![图片](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='2999' height='1687'></svg>)上海优质小学环线分布图跟北京优质资源高度集中在四个主城区不同的是，上海外环外也有约1/5的优质教育资源。

近两年外围区域也有很多不错的新建校，如雨后春笋般进入大家视野。

我在北京待过10年，来上海也有5年，对上海城市规划更均衡这点感触较深，这或多或少也能缓和一些教育资源的紧张。

# 上海多校划片之路——土壤环境 #

理解了一些政策的大原则，我们再回头看2020年。

今年是执行民办政策改革的第一年，在短期内暂未拿到效果的情况下，上海会不会全面推进多校划片，我个人持怀疑态度。

那什么时候会推进呢？能否有一些分析思路？

我们可以参考上海已执行多校划片的区域，有哪些特点，是否值得借鉴。

如前文所说，长宁全区小升初、徐汇部分小升初、静安部分小升初，已执行较长时间的多校划片，客观的讲，这些区域的教育资源相对来说方差较小。

长宁全区分为8号块，即便是延安初中、市三女中所在的1-3号块，由于小学、初中的集团化办学，区块间的差距也不是很大。

徐汇、静安执行小升初多校划片的部分区域，也具备同样的特点，即初中校间差异相对较小，如徐汇区天平路街道多校划片的位育初级和市二中学，如老静安多校划片的市西和同济七一，单纯从初中学校来看，差距并没有大家想象的那么大。

![图片](https://pica.zhimg.com/50/v2-6c743b72bc047ffafb3a116e362824f5_720w.jpg?source=2c26e567)![图片](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='2999' height='1687'></svg>)这里还有必要插入一个话题，公办初中的发展。

正如前文所讲，民办初中整体优于公办初中，很大部分原因是民办初中具备筛选生源的先天优势，使得优秀的小学毕业生更多地聚集到了民办初中，站在公办初中的立场来看，就是优质生源的流失，而这种情况大概率会在民办摇号之后得到改善，即优秀的小学毕业生会更多地聚集到其对口的公办初中，因此，公办初中生源结构的改善，使得教学质量有所提升，几乎是必然。

综上所述，执行多校划片的区域，其特点是教育资源相对均衡，那么，哪里具备这个特点，以及如何让更多区域具备多校划片的土壤和环境，我们可以持续观察。

另外，学区化、集团化，就是从根本上促进和塑造这种土壤。

政策与政策之间，是相互耦合，相辅相成的。

以上是关于多校划片，以及教育均衡化的观察和思考。

最后，我们再聊下，在这样的政策背景下，如何降低购买学区房的风险？

# 多校划片购房思路 #

有一点我们要明确，当下上海所有幼升小，以及绝大部分小升初都没有执行多校划片，且通过北京发布的多校划片政策来看，采用的是老人老办法、新人新办法的“不追溯”原则，即在政策颁布后的网签才生效，因此大可不必自己吓自己。现在已有的学区房，以及正打算购入的学区房，若参考北京的政策原则，是可以正常上学的。注意，这里强调的是网签时间，哪怕小孩政策颁布后出生，也不受影响。

[北京市西城区教育委员会关于西城区2020年义务教育阶段入学工作的实施意见_规范性文件_首都之窗_北京市人民政府门户网站](https://link.zhihu.com/?target=http%3A//www.beijing.gov.cn/zhengce/gfxwj/202005/t20200520_1903693.html)![图片](https://picx.zhimg.com/50/v2-df330fca9077995346fd2d6b2253368a_720w.jpg?source=2c26e567)![图片](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1488' height='588'></svg>)北京西城区多校划片政策另外，一贯制的学校在小升初阶段，不受多校划片影响（如长宁区建青实验），这样的一贯制优质校，会更加受人青睐。

![图片](https://picx.zhimg.com/50/v2-b78b533b4535ed0c79df366095660eb1_720w.jpg?source=2c26e567)![图片](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1488' height='380'></svg>)长宁区小升初8号块纵使多校划片多么骇人听闻，但有一件事并没有因此而改变——稀缺资源依然稀缺。

改变的是选择对象。过去是挑单校，若执行多校划片，那就是挑某个片区。

学区概念，依然存在。网络上传播的“学区房要凉了”大概率不会出现，要知道学区房的底层逻辑是供需矛盾，这点并不会因多校划片而改善。

值得注意的是，若某区域内的学校皆优质，那该片区便会成为稀缺资源，市场自然会给予对应的定价。

![图片](https://pic1.zhimg.com/50/v2-c2c6deea12f2f031c46f093df373d509_720w.jpg?source=2c26e567)![图片](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='3509' height='4962'></svg>)上海部分优质公办小学分布图实际上，很多家长朋友关注的便是房价，担忧学区溢价部分会因此被稀释，或被完全抽空。

我们的学区群里，也有家长提出：若我的房子降了200万，怎么办？

针对这个话题，我们也专门了解了下北京西城区的市场数据，来分享给大家。

西城区多校划片政策颁布后的市场变化，有涨有跌。

涨的情况：一种是优质学区聚集的区域，赶在政策前买进；一种是原对口一般的学区，周边有一所特别好的小学。

跌的情况：片区内普遍学区一般，优质校很少，此学区就跌了。

涨跌幅均在5%-10%（若要跌200万，那总价得2000万-4000万，而这远超学区房总价的中位数）。

我常说，教育投资和房产投资二者的损益不能放在一起对比。

过去很长一段时间，这二者都会带来超预期的回报，但如果我们必须要遵循自然法则的话——投入和产出对等，那么以一个能接受的房产投入，让下一代的教育有所保障，这是一个心态和选择，取决于每个为人父母的判断。
