下面说下 白云路小学 学生家长反馈的就读感受，和学校介绍，可供参考

1.学校简介

白云路小学位于国家级可持续发展实验区—西城区月坛地区，学校1960年建校，2014年改名为北京第二实验小学白云路分校。有近60名教师。是一所有良好育人环境，广受社会各方赞誉的学校。

2. 招生情况

一年级五到六个班，一个班40人左右。

3. 课程安排情况

每天都有体育课，每周两节英语，两节美术，两节音乐。

平常没有考试排名。

作业不多

4. 兴趣班情况

社团不收费，周一二五下午社团5:00放学。

一年级社团有美术，大合唱，象棋，魔方，剪纸，七巧板。

5. 对老师和学校整体的评价

老师不错，有问题会跟家长及时沟通。

整体感觉白小的作业不多，学习不是很紧张。

教学环境还可以，场地不大，就一个教学楼。

6. 作息情况

午休可以看课外书，可以画画。

学校有食堂，中午在学校吃饭。

平常最晚可以5点接孩子。

以上是从各方面收集的信息，简单白云路小学的介绍。
